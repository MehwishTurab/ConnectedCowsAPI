Monitor on the go

Mobile app that will provide real time data anout your livestock.

With sensors deployed on the field that provide real time data to the system which then stores the data in the main database of the system. After data is entered in the database it is then accessed the app and real time analysis is done and stats are displayed to the user accordingly.

Due to the use of IoT technologies the app can be used from anywhere in the world to view stats of the livestock in realtime without feeling any major delay in getting the stats from the system. Due to this you the user doesn't have to be on site to monitor your livestock. You can now do this anytime anywhere by just using the app.

NodeMCU

The main component in making the livestock monitoring smart.

This is the major component that is responisible for gathering all the data generated from the sensors and then saving it to the database. It has built in WiFi functionality which enables it to wirelessly transfer data over the internet without the hassel of internet cables.

The most important thing about the NodeMCU is that it is about the size of the palm of your hand which makes its small enough in size for easy deployment on site without any difficulties.

Node Express MongoDB Api

The components that are responsible for hardware software communication

We have used Node Express to develop our systems API's which are reponsible for telling the NodeMCU what to do with the data coming from the sensors and how to save the data in our databse. Without the proper usage of the api's the hardware becomes useless.

MongoDB is being used to provide database services for our system. It stores,collects,sends and recieves data from both our NodeMCU (the hardware) and the android application. The data saved in the database by the NodeMCU is accessed by the application is realtime to get livestock heal statistics.

Web Dashboard

A dashboard is in place to keep track of all the activites.

We have created a web dashboard in order to provide a control pannel for proper management of the complete system. It will also provide aspects of key performance indicators which will help us in order to further improve our applicaiton via updates in the future.

The dashboard is connected to our database which is MongoDB which allows the data on the dashboard to be updated constantly. The dashboard is kept in place to give us a snapshot of our system.

What is IoT?

IoT is short for Internet of Things. The Internet of Things refers to the ever-growing network of physical objects that feature an IP address for internet connectivity, and the communication that occurs between these objects and other Internet-enabled devices and systems.
